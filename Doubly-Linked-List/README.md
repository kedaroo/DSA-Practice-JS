# Doubly Linked Lists

| Method  | Time Complexity |
| ------- | --------------- |
| Shift   | O(1)            |
| Unshift | O(1)            |
| Push    | O(1)            |
| Pop     | O(1)            |
| Insert  | O(n)            |
| Remove  | O(n)            |
| Get     | O(n)            |
| Set     | O(n)            |
| forEach | O(n)            |
